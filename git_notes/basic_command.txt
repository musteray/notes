Create a git local repo
1. Git init 

Add to repo
1. Git add . or git add [filename]
2. Git commit -m "Your message here" 
or 
3. Git commit -am "Your message here" - When to use when simple changes

Checkout - get a specific file in your repo
git checkout -- [filename]

Remove a file
git rm [filename]

Rename and move 
git mv [old filename] space [new filename]

Status 
git status

Compare local to git local repo
Git diff 

Compare stage file to git local repo
git diff --staged

Log 
git log
git log --author="Specific name you want to see the log"
